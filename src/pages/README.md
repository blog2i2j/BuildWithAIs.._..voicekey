# pages/

路由页面组件目录。

## 文件

- `HomePage.tsx` - 首页仪表盘：显示 PTT 提示、历史统计汇总与区间趋势图。
- `SettingsPage.tsx` - 设置页：语言切换即时生效，其余设置走防抖自动保存；展示版本与更新状态，管理 ASR（含低音量模式）/手动 OpenAI-compatible 文本润色配置与连接测试/快捷键配置，支持开机自启后台启动开关、ASR 与润色 Key 明文显隐。自动保存状态在宽屏时固定显示在右侧悬浮卡片中，窄宽度下退回顶部提示；测试结果与校验提示继续就地展示，并提供日志查看入口。
- `HistoryPage.tsx` - 历史记录页：搜索/排序/分组展示，支持复制、删除与清空。
- `__tests__/SettingsPage.test.tsx` - 设置页加载/保存、连接测试与更新检查的渲染测试。
